what is website development?
Website development is simply the creation, build and maintenance of a website. This includes markup (the language developers use to create websites, such as HTML), coding (the programming language), network configuration and CMS (content management system). Many use "web development" to describe the actual construction of websites. It includes what you find "under the hood," or on the back end of a website, as well as any coding needed for the visible part of your website, or the "front end." <article> element.

Web development vs. web design
Website design is the process of creating the look and feel of a website. This includes the layout, color scheme, typography, and overall visual aesthetics. Designers use tools like Adobe Photoshop, Sketch, or Figma to create mockups and prototypes of the website's design. They also consider user experience (UX) and user interface (UI) principles to ensure that the website is easy to navigate and visually appealing. Web development and web design are two distinct but interconnected disciplines in the process of creating websites. Web development involves the technical implementation and functionality of a website. It encompasses the coding and programming required to bring the design to life. Web developers use technologies like HTML, CSS and JavaScript to build the user interface, while implementing back-end logic with programming languages like PHP, Python or Ruby. They may additionally tap into their familiarity with formats like XML files and JSON, which is often used for data interchange, to integrate data into the website. Web design, on the other hand, focuses on the visual aesthetics and user experience of a website. It involves crafting the layout, typography, color schemes, and overall visual appeal to create an engaging and intuitive user interface. Web designers may use tools like Photoshop, Illustrator and Figma to create mockups and find ways to improve the usability, accessibility and branding of a site.
