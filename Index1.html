Q1: Define HTML and explain its role in web application development

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It consists of a series of elements (tags) that define the content and structure of a webpage.

HTML is essential for web application development as it provides the basic framework for displaying content in a web browser. It allows developers to create headings, paragraphs, links, images, and other multimedia elements that make up a webpage.

HTML is often used in conjunction with CSS (Cascading Style Sheets) and JavaScript to create interactive and visually appealing web applications. While HTML provides the structure, CSS is used for styling and layout, and JavaScript adds interactivity and dynamic behavior.

In summary, HTML is the backbone of web development, enabling developers to create structured and accessible content that can be rendered by web browsers.

Q2: Describe the structure of an HTML document and explain the purpose of the following tags

  • <!DOCTYPE html>
    Declares the document type and version of HTML. Tells the browser that HTML5 is being used.
  • <html>
    The root element that wraps the entire content of the page, including the head and body.
  • <head>
    Contains meta-information about the page, such as title, character encoding, and linked CSS or JS.
  • <body>
    Contains the visible content shown to the user in the browser.
  • <title>
    Sets the title of the webpage displayed in the browser tab.
  • <meta>
    Defines metadata like character encoding, author, and viewport settings.

Q3: Explain the difference between HTML4 and HTML5. Provide two new features introduced in HTML5

HTML4 and HTML5 are versions of the HyperText Markup Language used to structure web content. Key differences include:

  • Doctype: HTML4 uses a long doctype, while HTML5 simplifies it with <!DOCTYPE html>.
  • Semantic Elements: HTML5 introduces tags like <header>, <footer>, <article>, and <nav> for better structure and meaning.
  • Multimedia Support: HTML5 includes native support for <audio> and <video> without requiring plugins like Flash.
  • Form Enhancements: HTML5 adds new input types (e.g., email, date) and attributes (e.g., placeholder, required) to improve forms.
  • APIs: HTML5 introduces powerful APIs such as Web Storage, Geolocation, and Canvas.
  • Accessibility: Emphasizes better support for assistive technologies using semantic structure.

New features introduced in HTML5 include:

  • Local Storage: Allows web apps to store data in the browser for offline use, unlike cookies.
  • Canvas Element: Enables drawing 2D graphics and animations using JavaScript.
  • Audio and Video Support: Native tags for embedding multimedia without external plugins.
